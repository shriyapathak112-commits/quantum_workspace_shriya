# Portfolio Optimization basics

Portfolio optimization is the process of selecting the best proportions of various assets to maximize expected returns for a given level of risk, or to minimize risk for a given level of expected return.

#### Modern Portfolio Theory (MPT)

* Risk vs. Return: Assets are evaluated based on their expected return (mean) and variance (risk).
* Efficient Frontier: The set of optimal portfolios that offer the highest expected return for a defined level of risk.
* Diversification: Combining assets with low or negative correlations to reduce overall portfolio variance.

#### Quantum Application

This combinatorial problem can be mapped directly to a Quadratic Unconstrained Binary Optimization (QUBO) problem, which can be solved efficiently on near-term hardware using the Quantum Approximate Optimization Algorithm (QAOA) or Quantum Annealing.
