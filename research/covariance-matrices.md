# Covariance Matrices

A covariance matrix is a mathematical matrix that quantifies how pairs of financial assets move together. It is a critical input parameter for calculating portfolio risk in optimization algorithms.

#### Key Properties

* Symmetry: The matrix is square and symmetric, as the covariance between Asset A and Asset B equals the covariance between Asset B and Asset A.
* Positive Semi-Definite: Ensures that the calculated portfolio variance is always non-negative.
* Diagonal Values: The diagonal elements of the matrix represent the individual variance (risk) of each asset.

#### Quantum Application

Quantum computers utilize Quantum Principal Component Analysis (QPCA) or quantum-enhanced kernel methods to process and calculate distance metrics in high-dimensional covariance matrices faster than classical architectures.
